Enhancement No 1

The first proposed enhancement for the website is to highlight the selected menu to improve the usability and accessability. So that user will know which part of the page they are taking actions

Implementation

In order to achive this functionality, we need to do follows:

  1. import JQUERY library in our header of html
  2. implement functionality to append the class name for active section in our Javascript file. And trigger the function when page is loaded
  3. create active class in our css file to distinguish navigation menu

Futher instruction coud be found in the following links:

  • JQUERY Api documentation
  • Stackoverflow query (Navbar highlight for current page)

Note: You can find the output of implementation in this Enhancement home, which is the current active menu is highlighted page

Enhancement No 2

The second proposed enhancement for the website is to control the time where user will have limited to to appy. This will improve the user experience.

Implementation

In order to achive this functionality, we need to do follows:

  1. We need to create startTimer function, where it will count down in our Javascript file. And trigger the function when page is loaded
  2. Show timer on the HTML file where user applies for the job
  3. create neccessary class name (we will call it timer)
